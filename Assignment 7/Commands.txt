Router 5 Configuration After creating the topology, we’ll click router for configuration.
Router 5- CLI:

en
config t

interface Se2/0
ip address 100.1.1.1 255.255.255.0
no shut

interface Se3/0
ip address 20.1.1.1 255.255.255.0
no shut

router ospf 1
router-id 1.1.1.1
network 100.1.1.0 0.0.0.255 area 0
network 20.1.1.0 0.0.0.255 area 0
end
