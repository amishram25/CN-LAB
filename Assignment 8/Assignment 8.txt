Router 0 Configuration After creating the topology, we’ll click router for configuration.

Router 0- CLI:

en
config t

interface Se2/0
ip address 10.0.0.1 255.255.255.252
no shut

interface Fa0/0
ip address 192.168.1.1 255.255.255.0
no shut

router ospf 1
router bgp 100
neighbor 10.0.0.2 remote-as 200
network 192.168.1.0 mask 255.255.255.0
end
